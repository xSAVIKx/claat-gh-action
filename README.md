# `claat` action

A GitHub action with the [`claat`](https://github.com/googlecodelabs/tools/tree/main/claat) tool

## Inputs

## `args`

**Required** The `claat` CLI arguments to be passed.

## Outputs

## `out`

The output of the `claat` CLI command.

## Example usage

```yaml
uses: xSAVIKx/claat-gh-action
with:
    args: 'version'
```
